# 🎯 BIENVENUE - Votre Site React ONE DAY ASBL est Prêt!

## 🚀 Pour Vois le Site Maintenant (5 min)

⏱️ **Suivez ces 3 étapes EXACTES:**

### 1️⃣ Préparer (1 min)
Ouvrez ce dossier:
```
d:\ONE DAY ASBL\Administratif doc\site one day asbl\frontend\
```

Renommez:
- `index_new.html` → `index.html`

*(Le ancien `index.html` peut être archivé ou supprimé)*

### 2️⃣ Installer (3 min)

Ouvrez **PowerShell** à cet endroit :
```
Ctrl + Maj + Clic Droit
→ "Ouvrir PowerShell ici"
```

Tapez:
```bash
npm install
```

⏳ **Attendez 3 minutes** - Vous verrez "added XXX packages"

### 3️⃣ Lancer (30 sec)

```bash
npm run dev
```

✨ **BRAVO!** Votre site s'ouvre automatiquement sur:
```
http://localhost:3000
```

## 🆘 Problèmes?

### ❌ "npm: command not found"
→ Installez Node.js: https://nodejs.org/

### ❌ Port 3000 occupé
→ Utilisez un autre port:
```bash
npm run dev -- --port 3001
```

### ❌ Site blanc
→ Appuyez sur **F12** et vérifiez les erreurs rouges en bas

### ❌ Autres problèmes?
→ Lisez **INSTALLATION_CHECKLIST.md**
